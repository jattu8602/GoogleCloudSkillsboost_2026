## 🤝 How to Contribute

We welcome contributions! However, we have a strict policy to ensure the quality of scripts.

> [!IMPORTANT]
> **Mandatory Script**: Every contribution MUST include a `jattu.sh` script.
> **Tested Only**: We only accept well-tested shell scripts.
> **No Hacking**: Malicious or hacking scripts are strictly prohibited.

For more details, see [CONTRIBUTING.md](./CONTRIBUTING.md).
