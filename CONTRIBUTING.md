# Contributing to Google Cloud Skillsboost 2026

Welcome to the **GC_2026** community! We are excited to have you contribute to this repository. To maintain the quality and security of our scripts, please follow these guidelines.

## Contribution Rules

> [!IMPORTANT]
> **Mandatory Script Requirement**
> All Pull Requests (PRs) **MUST** contain a script named `jattu.sh` or integrate with the existing `jattu.sh` framework. PRs without a `jattu.sh` script will not be merged.

### Technical Requirements
- **Well-Tested Scripts**: Only well-tested shell scripts are accepted. Please provide proof of testing in your PR description.
- **Security First**: Any scripts that could be used for hacking, unauthorized access, or malicious activities are strictly prohibited and will be rejected.
- **Shell Scripts Only**: Focus on high-quality shell scripts for Google Cloud Arcade labs and tasks.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or lab solution.
3. Ensure your contribution includes the mandatory `jattu.sh` script or follows its structure.
4. Test your scripts thoroughly on Google Cloud Shell.
5. Submit a PR with a clear description of your changes.
